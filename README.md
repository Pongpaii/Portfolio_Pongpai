# Pongpai Sodsong — Portfolio

Personal portfolio website built with Next.js, TypeScript, Framer Motion, and Tailwind CSS.

## Stack
- **Frontend**: Next.js 14 (App Router) + TypeScript
- **Animations**: Framer Motion + GSAP
- **Styling**: Tailwind CSS + CSS Variables
- **Deploy**: Vercel

## Run locally
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy
Push to GitHub → connect repo to [Vercel](https://vercel.com) → auto-deploy on every push.
